A fully client-side personal expense tracker that runs entirely in the browser with no backend or server required. Data is persisted locally to your device.

## Features

- Add and delete income and expense transactions
- Category management with custom icons
- Interactive charts and spending breakdown via Recharts
- Smooth animations with Framer Motion
- Date formatting and filtering
- Fully offline — works without internet

## Stack

- **Framework**: React 19, Vite
- **Styling**: Tailwind CSS v4
- **Charts**: Recharts
- **Animations**: Framer Motion
- **Routing**: React Router v7
- **Storage**: Local file-based persistence
